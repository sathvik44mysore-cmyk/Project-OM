# Project OM: Omni-Agent Learning System

Project OM is a comprehensive AI engineering project designed to explore Multi-Agent Systems, the Model Context Protocol (MCP), and custom Neural Network integration.

## Goals
- **Multi-Agent Orchestration:** Building a team of specialized agents that collaborate.
- **MCP Integration:** Connecting agents to real-world tools like GitHub, Reddit, and Web Search.
- **Deep Learning:** Developing a custom PyTorch model for intent classification.
- **Portfolio Ready:** A structured, documented, and professional-grade AI project.
